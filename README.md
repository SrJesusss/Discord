# This mod adds:

- **50** Discord Jokers (All the Jokers on this mod)

- **24** Discord Users (Users who asked me to add them to this mod or i simply added them)

- **4** Food Jokers (Jokers related to food)

- **5** Crossmod Jokers (Jokers that requieres this mod and another to appear)
